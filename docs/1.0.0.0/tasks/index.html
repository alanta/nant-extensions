NAnt-Extensions Help Task Reference v1.0.0.0

Task Reference

[This is preliminary documentation and subject to change.]

Task Summary
mbunit Runs MbUnit tests.
mbunit-initcounters Initializes all MbUnit counters with a value of 0. Existing property values will be overwritten.
mspec Runs Machine.Specifications tests.
mspec-initcounters Initializes all Machine.Specifications counters with a value of 0. Existing property values will be overwritten.
tc-addstatistic Adds a single TeamCity build statistics value to teamcity-info.xml.
tc-addstatistic-fromprops Adds TeamCity build statistics values to teamcity-info.xml based on NAnt properties matching the specified prefix.
tc-addstatistic-list Adds TeamCity build statistics values to teamcity-info.xml.
tc-appendstatustext Appends a status message to the TeamCity build status to teamcity-info.xml.
tc-buildstatus Reports the build status to TeamCity.
tc-progress Reports the build progress to TeamCity.
