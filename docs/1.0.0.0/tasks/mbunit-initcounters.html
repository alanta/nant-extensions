<mbunit-initcounters>

[This is preliminary documentation and subject to change.]

Initializes all MbUnit counters with a value of 0. Existing property values will be overwritten.

The <mbunit> task updates these NAnt properties when the test run has completed:
  • mbunit.asserts - The number of asserts.
  • mbunit.failures - The number of failed tests.
  • mbunit.ignored - The number of ignored tests.
  • mbunit.run - The number of tests run.
  • mbunit.skipped - The number of skipped tests.
  • mbunit.successes - The number of successful test.
