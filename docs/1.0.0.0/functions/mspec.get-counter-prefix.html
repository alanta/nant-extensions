mspec::get-counter-prefix

[This is preliminary documentation and subject to change.]

Returns the prefix for the counters that are initialized by the <mspec-initcounters> task and updated by the <mspec> task.

Usage

string mspec::get-counter-prefix()

Return Value

The prefix string.

Examples

  • <echo message="The prefix for the Machine.Specifications counters is: ${mspec::get-counter-prefix()}" />

Requirements

Assembly: NAntExtensions.Machine.Specifications (1.0.0.0)
